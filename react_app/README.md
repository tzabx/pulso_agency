# Pulso Agency React

Implementacion paralela en React + Vite de la misma landing que existe en Astro.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Datos compartidos

Este proyecto consume los mismos JSON que Astro desde:

- `../shared/data/home.json`
- `../shared/data/site.json`
